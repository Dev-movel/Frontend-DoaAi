# 📱 Frontend DoaAi

> Aplicativo mobile desenvolvido em **Flutter** para conectar pessoas que desejam doar com quem precisa.

---

## 📌 Sobre o projeto

O **DoaAi** é uma plataforma com foco social que facilita a doação de itens entre usuários.

Este repositório contém o frontend mobile, desenvolvido em Flutter, com uma interface moderna, responsiva e focada na experiência do usuário.

---

## ✨ Funcionalidades

- 📦 Listagem de itens disponíveis para doação  
- 🔎 Busca e filtro de doações  
- 👤 Cadastro e login de usuários  
- 📍 Geolocalização de doadores/receptores  
- 💬 Interface intuitiva e responsiva  
- 🔗 Integração com API backend

## 🚀 Como rodar o projeto

### 📥 Clone o repositório
```bash
git clone https://github.com/Dev-movel/Frontend-DoaAi.git
```


### 📦 Instale as dependências
```bash
flutter pub get
```


### ▶️ Rode o projeto
```bash
flutter run
```
